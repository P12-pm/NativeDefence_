Course Overview

The Native Defence Threat Hunting Professional course is designed for SOC analysts and security engineers who want to master the art of manual threat detection. While SIEMs and EDRs are powerful, they only catch what they know.

In this course, you will learn how to formulate hypotheses, analyze network traffic patterns, and dig deep into endpoint memory forensics to find the "unknown unknowns." You will work on a live cyber range simulating a compromised enterprise network.

What You Will Learn

  • Threat Hunting Methodologies (The loop)
  • IOC vs IOA Detection
  • Network Traffic Analysis (Wireshark/Zeek)
  • Memory Forensics (Volatility)
  • Windows Event Log Deep Dives
  • Automating Hunts with Python

Curriculum

  • The Hunter's Mindset vs The Responder's Mindset
  • Cyber Kill Chain & MITRE ATT&CK Framework
  • Setting up the Hunting Lab (ELK Stack)

  • Hunting for Persistence Mechanisms (Registry/Cron)
  • Process Injection & Hollowing Detection
  • Powershell De-obfuscation and Analysis

  • Beaconing Analysis with Zeek
  • Detecting C2 over DNS and HTTPS
  • Lateral Movement detection (SMB/RDP)

Real-world scenario: A financial institution has been compromised. You are given access to the logs and memory dumps. Find the intruder, map their movement, and write the report.

Prerequisites

To ensure you get the most out of this course, we recommend:

  • Basic understanding of Networking (TCP/IP, DNS).
  • Familiarity with Windows and Linux Command Line.
  • Previous experience in IT or Junior SOC role is preferred.
